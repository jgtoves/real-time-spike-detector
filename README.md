# esp8266-real-time-spike-detector
An app that cross references radio signals to find summations that can determine if there are rogue devices around. 
<br>
<br>
• git clone https://github.com/jgtoves/esp8266-real-time-spike-detector
<br>
• pkg install termux-api python
<br>
• pip install statistics
<br>
• For Bluetooth, simply run: termux-bluetooth-scan, and if more than a few ID's show up, then obviously we have company.
<br>
# Requires termux!
# Dependencies: scapy
